Features

Everything you need to
audit your system

Appnort combines AI intelligence with local-first privacy to give you complete visibility into your installed software.

AI-Powered Categorization

Uses Groq's Llama 3.3 70B model to intelligently categorize unknown applications into 8 distinct categories. Falls back to rule-based matching when offline.

Groq API Llama 3.3 Smart Caching

Security Risk Assessment

Every application is rated Low, Medium, or High risk. Identifies potential threats like malware, RATs, and cracked software.

PDF Audit Reports

Generate professional, color-coded PDF reports with category breakdowns, risk levels, and publisher details in one click.

Registry Deep Scan

Scans both 64-bit and 32-bit registry paths to detect every installed program, including hidden and system-level software.

Local-First Privacy

All data stays on your machine. API keys stored locally. Smart caching minimizes external calls. No accounts needed.

Dark & Light Themes

Modern CustomTkinter UI with system, dark, and light theme options. Clean interface that's easy on the eyes.

How It Works

From scan to report
in seconds

01

Scan Programs

Click the scan button. Appnort reads your Windows Registry to detect every installed application.

02

Rule-Based Check

Known applications are instantly categorized using built-in keyword matching across 8 categories.

03

AI Classification

Unknown apps are sent to Groq AI in batches. Each gets a category and security risk rating.

04

Export Report

View results in the UI or export a professional, color-coded PDF audit report instantly.
